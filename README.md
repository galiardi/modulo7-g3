# Modulo 7 - Grupal 3

1. Clonar repositorio:

```sh
git clone https://github.com/galiardi/modulo7-g3
```

2. Instalar dependencias:

```sh
cd modulo7-g3
```

```sh
npm install
```

3. Levantar servidor:

```sh
npm run dev
```

4. Visitar <a href="http://localhost:3000" target="_blank">localhost:3000.</a>
